# ⚡ PokeDev - Centro Pokémon Administrativo

Uma aplicação Full Stack completa para gerenciamento de treinadores e coleções de Pokémons, desenvolvida como projeto prático de desenvolvimento web.


## 🚀 Tecnologias Utilizadas

### Frontend
* **Next.js 14** (App Router)
* **Tailwind CSS** (Estilização)
* **Axios** (Consumo de API)
* **TypeScript**

### Backend
* **NestJS** (Framework Node.js)
* **TypeORM** (ORM para Banco de Dados)
* **PostgreSQL** (Hospedado no Neon.tech)
* **JWT** (Autenticação Segura)

## 🛠️ Funcionalidades
* **Autenticação:** Sistema de registro e login de treinadores com proteção de rotas.
* **Captura Aleatória:** Integração com a PokéAPI para capturar Pokémons com atributos randômicos.
* **CRUD de Pokedex:** Listar, editar (subir nível) e soltar Pokémons da sua coleção.
* **Deploy Automático:** Frontend na **Vercel** e Backend no **Render**.

## 💻 Como rodar o projeto localmente

### Pré-requisitos
* Node.js instalado
* PostgreSQL rodando ou uma URL do Neon.tech
